# Unified Model

Forecasting warm dark matter (WDM) constraints from joint dwarf galaxy and star cluster populations predicted for LSST. Built on Nadler et al.'s Census forward-modeling framework. Fitting parametric models to simulated distributions to infer WDM particle mass lower bounds.
